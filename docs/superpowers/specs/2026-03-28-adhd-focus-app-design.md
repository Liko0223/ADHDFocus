# ADHD Focus App — 设计文档

## 概述

一款面向 ADHD 用户（首先是设计师群体）的 macOS 原生专注辅助工具。通过"模式"系统管理应用和网站的访问权限，结合番茄钟和环境配置，帮助用户进入并维持专注状态。

**目标用户：** 先为自己使用（UI 设计师，有 ADHD），验证后面向设计师群体推广。

**设计调性：** 设计师美学 — 精致、有品位，应用本身就是一件设计作品。

## 应用形态

- **菜单栏 + 主窗口** 结合
  - 菜单栏：快速切换模式、查看番茄钟倒计时、当前状态
  - 主窗口：模式编辑器、专注统计仪表盘、设置页面
- **技术栈：** 原生 SwiftUI，macOS 平台

## 核心概念：模式（Focus Mode）

一切围绕"模式"展开。每个模式是一个独立的配置包，定义了在该工作状态下的所有规则。

### 数据模型

```
FocusMode
├── 基本信息
│   ├── 名称
│   ├── 图标
│   └── 统计标签（用于归类专注时间）
├── 应用规则
│   ├── 允许的应用列表
│   ├── 禁止的应用列表
│   └── 未列出的应用 → 默认策略（允许 / 提醒 / 禁止）
├── 浏览器规则（当浏览器在允许列表中时生效）
│   ├── 允许的 URL 列表（如 dribbble.com, behance.net）
│   ├── 禁止的 URL 列表（如 weibo.com, xiaohongshu.com）
│   └── 未列出的 URL → 默认策略（允许 / 提醒 / 禁止）
├── 限制策略
│   ├── 严格程度：温和提醒 / 强制退出 / 延迟允许（5分钟缓冲）
│   └── 冷却期：退出模式后需等待 X 分钟才能重新进入（防止反复开关）
├── 番茄钟配置
│   ├── 工作时长（默认 25 分钟，用户可自定义）
│   ├── 休息时长（默认 5 分钟，用户可自定义）
│   └── 长休息时长（每 4 轮后，默认 15 分钟，用户可自定义）
└── 环境配置
    ├── 自动开启 macOS 勿扰模式
    └── 自动隐藏 Dock
```

### 预设模式

1. **深度设计** — 白名单：Figma/Sketch/PS + 浏览器（仅设计网站），严格限制社交和无关网站
2. **调研灵感** — 白名单：浏览器（更宽泛的 URL 白名单）+ 设计工具，温和提醒社交应用
3. **沟通协作** — 白名单：微信/飞书/Slack + 浏览器，番茄钟可关闭
4. **写作整理** — 白名单：Notion/备忘录/Markdown 编辑器，严格限制其他

用户可修改预设，也可创建新模式。

## 系统架构

应用由 4 个组件构成：

### 1. 菜单栏组件
- 一键切换模式（网格卡片展示所有模式）
- 当前模式状态 + 番茄钟倒计时进度条
- "结束专注"按钮
- "打开主窗口"入口

### 2. 主窗口
- **模式编辑器：** 创建、编辑、删除模式，配置应用规则/URL 规则/番茄钟/环境
- **专注统计仪表盘：** 今日概览（总时长、番茄钟数、拦截次数、连续天数）
- **设置页面：** 全局配置、权限管理

### 3. 后台服务
- 使用 `NSWorkspace.didLaunchApplicationNotification` 监听应用启动
- 检测到黑名单应用时通过 `NSRunningApplication.terminate()` 强制退出
- 控制 macOS 勿扰模式（通过 Shortcuts/AppleScript）
- 记录专注数据到 SwiftData

### 4. Chrome Extension
- 读取当前模式的 URL 规则
- 拦截不允许的网站，展示自定义拦截页面
- 拦截页包含：原因说明、番茄钟剩余时间、引导去白名单网站、"允许 5 分钟"弹性选项
- 与主应用通信（Native Messaging）

### 数据层
- **SwiftData** — 存储模式配置和专注记录，数据全部在本地
- **Native Messaging** — 主应用与 Chrome Extension 之间的配置同步

## 用户交互流程

### 应用拦截
1. 用户在专注模式下尝试打开被禁应用
2. 后台服务检测到应用启动
3. 强制退出该应用
4. 发送系统通知：告知原因、番茄钟剩余时间、提供"切换模式"选项

### URL 拦截
1. 用户在浏览器中访问被禁网站
2. Chrome Extension 检测 URL 匹配黑名单
3. 展示自定义拦截页面，引导去白名单内的网站（如 Dribbble、Behance）
4. 提供"允许 5 分钟"弹性选项

### 番茄钟
1. 进入模式时番茄钟自动开始（或手动启动）
2. 工作时段：当前模式的限制规则生效
3. 休息时段：限制自动解除，可自由使用所有应用
4. 每 4 轮工作后进入长休息
5. 番茄钟结束时发送系统通知

## 专注统计

### 今日概览（MVP）
- 总专注时长
- 完成的番茄钟数量
- 被拦截的应用/网站次数
- 当前连续专注天数（streak）

### 后续迭代
- 每日/每周专注时长趋势图
- 各模式使用占比
- 最常被拦截的应用/网站 Top 5
- 分心时间段分析

## macOS 权限需求

- **辅助功能权限（Accessibility）** — 监听和控制其他应用
- **通知权限** — 番茄钟提醒、拦截通知
- **勿扰模式控制** — 通过 Shortcuts/AppleScript

## MVP 范围

### 包含
1. 菜单栏 + 主窗口基础框架
2. 4 个预设模式 + 自定义模式编辑
3. 应用黑名单监控 & 强制退出
4. Chrome Extension URL 拦截
5. 番茄钟（用户可自定义工作/休息/长休息时长）
6. 基础专注统计（今日概览）
7. 自动开启/关闭勿扰模式

### 不包含（后续迭代）
- 白噪音/氛围音
- 自动化触发（打开 Figma 自动进入模式）
- 趋势图表和分心洞察
- Safari Extension
- 桌面壁纸自动切换
- iCloud 同步
